# cncf-excting-projects
2021 CNCF Open Source Projects that we should try

| Project Name  | Category | Description | Note |
| ------------- | ------------- | ------------- | -------------|
| [Schemahero](https://github.com/schemahero/schemahero)  | Database | A Kubernetes Operator for Declarative Schema Management for various databases. | Support RDS, Google CloudSQL, etc.
| [Strimzi](https://strimzi.io/)  | Streaming & Messaging  | Provides a way to run an Apache Kafka cluster on Kubernetes in various deployment configurations. | ------------- |
| [Pravega](https://pravega.io/)  | Streaming & Messaging  | Open source distributed storage service implementing Streams. | A high-performance, durable, elastic, and unlimited append-only byte stream with strict ordering and consistency. |
| [Tremor](https://github.com/tremor-rs/tremor-runtime)  | Streaming & Messaging  | An event processing system which was originally designed as a replacement for software such as [Logstash](https://www.elastic.co/products/logstash) or [Telegraf](https://www.influxdata.com/time-series-platform/telegraf/). |  ------------- |
| [Backstage](https://github.com/backstage/backstage)  | Application Definition & Image Building  | Open platform for building developer portals | Backstage was created by Spotify but is now hosted by the Cloud Native Computing Foundation (CNCF) as a Sandbox level project.  |
| [Kudo](https://github.com/kudobuilder/kudo)  | Application Definition & Image Building  | Provides a declarative approach to building production-grade Kubernetes Operators covering the entire application lifecycle. | ------------- |
| [ArtifactHub](https://github.com/artifacthub/hub) | Application Definition & Image Building | A web-based application that enables finding, installing, and publishing packages and configurations for CNCF projects. | Include Helm charts and plugins, Falco configurations, Open Policy Agent (OPA) policies, OLM operators, Tinkerbell actions, kubectl plugins, Tekton tasks, KEDA scalers and CoreDNS plugins.
| [KubeVirt](https://github.com/kubevirt/kubevirt)  | Application Definition & Image Building | A virtual machine management add-on for Kubernetes | Aim is to provide a common ground for virtualization solutions on top of Kubernetes. |
| [Porter](https://github.com/getporter/porter) | Application Definition & Image Building | Package your application artifact, client tools, configuration and deployment logic together as a versioned bundle that you can distribute, and then install with a single command. | -------------|
| [Telepresence](https://github.com/telepresenceio/telepresence/tree/release/v2) | Application Definition & Image Building | A fast, efficient local development for Kubernetes microservices. | -------------|
| [Brigade](https://github.com/brigadecore/brigade) | Application Definition & Image Building. | Script simple and complex workflows using JavaScript. | -------------|
| [keptn](https://github.com/keptn/keptn) | Application Definition & Image Building. | An event-based control plane for continuous delivery and automated operations for cloud-native applications. | -------------|
| [Openmetrics](https://github.com/OpenObservability/OpenMetrics) | Observability | A specification built upon and carefully extending Prometheus exposition format in almost 100% backwards-compatible ways. | -------------|
| [trickster](https://github.com/tricksterproxy/trickster) | Observability | An HTTP reverse proxy/cache for http applications and a dashboard query accelerator for time series databases.| -------------|
| [OpenTelemetry](https://github.com/open-telemetry) | Observability | Portable telemetry a built-in feature of cloud-native software. | -------------|
| [Chaos Mesh](https://github.com/chaos-mesh/chaos-mesh) |  Chaos Engineering  | Cloud-native Chaos Engineering platform that orchestrates chaos on Kubernetes environments. | -------------|
| [k3s](https://github.com/k3s-io/k3s) | Edge Computing | Lightweight Kubernetes. Production ready, easy to install, half the memory, all in a binary less than 100 MB. | -------------|
| [Crossplane] | Cloud Infrastructure Management | An open source Kubernetes add-on that enables platform teams to assemble infrastructure from multiple vendors, and expose higher level self-service APIs for application teams to consume.  | -------------|
| [Volcano](https://github.com/volcano-sh/volcano) | Batch Scheduling System | A batch system built on Kubernetes. | Integrates with generalized domain frameworks like TensorFlow, Spark, PyTorch, MPI, etc,|
| [BFE](https://github.com/bfenetworks/bfe) | Networking | A modern layer 7 load balancer from baidu. | -------------|
| [Service Mesh Interface](https://github.com/servicemeshinterface/smi-spec) | Service Mesh Interface | A specification for service meshes, with a focus on those that run on Kubernetes. | -------------|
| [Open Serviec Mesh](https://github.com/openservicemesh/osm) | Networking | A lightweight, extensible, Cloud Native service mesh that allows users to uniformly manage, secure, and get out-of-the-box observability features for highly dynamic microservice environments. | -------------|
| [Kuma](https://github.com/kumahq/kuma) | Networking | A modern Envoy-based service mesh that can run on every cloud, in a single or multi-zone capacity, across both Kubernetes and VMs.  | -------------|
| [Network Service Mesh](https://github.com/networkservicemesh) | Networking | ------------- | -------------|
| [CNI-Genie](https://github.com/cni-genie/CNI-Genie) | Networking | Enables container orchestrators (Kubernetes, Mesos) to seamlessly connect to the choice of CNI plugins installed on a host. | -------------|
| [Kube-OVN](https://github.com/kubeovn/kube-ovn) | Networking | A CNCF Sandbox Level Project, integrates the OVN-based Network Virtualization with Kubernetes. | -------------|
| [Longhorn](https://github.com/longhorn/longhorn) | Storage | A distributed block storage system for Kubernetes. | -------------|
| [OpenEBS](https://openebs.io/) | Storage | The most widely deployed and easy to use open-source storage solution for Kubernetes. | -------------|
| [ChubaoFS](https://github.com/chubaofs/chubaofs) | Storage | A cloud-native storage platform that provides both POSIX-compliant and S3-compatible interfaces. | -------------|
